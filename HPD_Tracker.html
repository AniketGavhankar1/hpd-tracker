        ☁ Cloud Sync Settings

        Choose a sync provider. No Azure or org permissions required.

        Setup (2 minutes, free):
        1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
        2. Click Generate new token (classic)
        3. Give it a note (e.g. HPD Tracker), tick scope: gist
        4. Copy the token and paste below
        5. Leave Gist ID blank for first use — it creates one automatically
        ☁ Auto-sync is ON — every save & delete is pushed to Gist automatically. Data loads from Gist when the page opens.
        Setup (1 minute, free):
        1. Go to jsonbin.io → Sign up (free)
        2. Dashboard → API Keys → copy your Secret Key
        3. Paste it below — leave Bin ID blank for first use
        ☁ Auto-sync is ON — every save & delete is pushed to JSONBin automatically. Data loads from JSONBin when the page opens.
